# globaltech

## Instalação

1. Clone o repositório:
  `git clone https://github.com/herverson/globaltech.git`

2. Navegue até o diretório do projeto:
`cd globaltech`

3. Instale as dependências:
`flutter pub get`

4. Execute o aplicativo:
`flutter run`
